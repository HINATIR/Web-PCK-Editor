# Web PCK Editor

ブラウザでPCKファイルを編集できるWebツールです。

## 機能  
ブラウザ上でテクスチャやオーディオファイルを含まない純粋なスキンパックの編集、作成を可能にします。

## 使用方法

[WebPCKEditorを開く](https://HINATIR.github.io/Web-PCK-Editor/WebPCKEditor.html)
